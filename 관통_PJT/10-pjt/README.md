# 📌 프로젝트 개요

**TravUs 프로젝트의 전체 기획·설계 명세서**를 완성하기 위해
다음과 같은 작업을 진행하였다.

TravUs는 **무장애 여행(Barrier-Free) 추천 서비스**로,
사용자 맞춤 여행지 추천, 접근성 정보 제공, AI 카메라 기반 설명 기능,
지역·카테고리 기반 여행지 조회 등을 제공하는 서비스이다.

명세서 요구에 따라

* 요구사항 도출
* WBS & 간트차트
* 유스케이스 다이어그램
* ERD
* 화면 설계
* API 명세서

를 모두 정리하였다.

* 향후 여행지 데이터 및 AI 기능 확장성

## 3️⃣ **ERD(Entity Relationship Diagram) 설계**

### 테이블 구성 (총 10개)

| 테이블                  | 설명            |
| -------------------- | ------------- |
| User                 | 사용자 정보        |
| TravelSpot           | 여행지 기본 정보     |
| AccessibilityInfo    | 무장애 정보        |
| TravelSpotCategory   | 카테고리(대/중/소분류) |
| Course               | 여행 코스         |
| CourseSpot           | 코스 내 여행지      |
| Bookmark             | 즐겨찾기          |
| AIRecognitionLog     | AI 인식 결과 기록   |
| AIRecognitionMessage | TTS 변환 메시지    |
| UserPreference       | 접근성/사용자 유형 정보 |
